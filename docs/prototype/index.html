A Announcement Bar AppUI prototype

Prototype — UI source of truth

Static HTML/CSS mockups describing the target interface for the Announcement Bar App. The real admin (frontend/) and storefront bar must look equivalent to these screens. All screens share one stylesheet, styles.css, so colors, spacing and radius stay consistent. Model: many saved bars per shop, at most one active at a time; the storefront shows the single active bar.

☰ Admin — bars list

The landing list of saved bars with an Active badge on the one enabled bar, plus Edit/Delete per row. admin-list.html

 → ✎ Admin — bar editor

Per-bar form: title, enabled toggle, message, colors, countdown, contextual save bar and a live preview. admin-editor.html

A mock storefront with the single active bar prepended at the top, ticking its countdown. storefront-bar.html
